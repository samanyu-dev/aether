# Security Policy

## Supported Versions

We actively support and patch the following versions of Aether packages:

| Version | Supported |
| :--- | :--- |
| **v0.1.0-beta** | ✔️ Active Support |
| < v0.1.0 | ❌ Not Supported |

## Reporting a Vulnerability

We take the security of Aether extremely seriously. If you discover any security vulnerability, please do **NOT** report it publicly via a GitHub Issue. 

Instead, please report security vulnerabilities directly to our maintainers at **security@aether-observe.org**. We will acknowledge your report within 24 hours and provide a detailed timeline for coordinate disclosure and patching.
